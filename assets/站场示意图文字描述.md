# 铁路中间站场示意图文字描述
## 一、站场整体布局
本站为标准铁路中间站，行车基准：下行方向线路从左至右，上行方向线路从右至左；中间Ⅱ道（IIG）为正线，上方3道（3G）、下方1道（1G）为两条到发侧线，1道左侧尽头设置专用安全线；站场上行咽喉区段存在6‰连续下坡。
全站划分为三大区域：左侧下行咽喉区、中部股道区、右侧上行咽喉区。

## 二、轨道区段清单
| 区段编号 | 区段位置 & 功能说明 |
| ---- | ---- |
| JXG | 下行接近区段，下行进站信号X外方区间线路 |
| IIAG | 下行进站内方区段，X信号机至1号道岔之间正线 |
| 3G | 3道到发线，上方侧线停车区段 |
| IIG | Ⅱ道正线到发线，车站主线通过区段 |
| 1G | 1道到发线，下方侧线停车区段 |
| 安全线 | 1道左下方尽头线路，末端配备车挡，用于防溜避险 |
| IIBG | 上行进站内方区段，4号道岔至上行进站S之间正线 |
| JSG | 上行接近区段，上行进站信号S外方区间线路 |

## 三、道岔设备清单
| 道岔编号 | 安装位置 | 线路分支功能 |
| ---- | ---- | ---- |
| 1#道岔 | 下行咽喉IIAG末端 | 直向连通Ⅱ道正线，侧向分岔连接3道、安全线方向 |
| 3#道岔 | 3道下行入口 | 负责3道股道与正线的连通切换 |
| 4#道岔 | 上行咽喉IIBG始端 | 直向连通Ⅱ道正线，侧向分岔连接3道、1道方向 |
| 2#道岔 | 1道上行出口 | 负责1道股道与正线的连通切换 |
| 5#道岔 | 安全线入口处 | 控制安全线的接入与隔离 |

## 四、全站信号机分类说明
### 1. 进站信号机（高柱主体信号）
1. X：下行进站信号机，分界JXG、IIAG区段，防护下行进站进路
2. S：上行进站信号机，分界JSG、IIBG区段，防护上行进站进路

### 2. 股道出站信号机
#### 下行出站（股道右端，发往上行区间）
- X3：3道下行出站信号
- XII：Ⅱ道下行出站信号
- X1：1道下行出站信号

#### 上行出站（股道左端，发往下行区间）
- S3：3道上行出站信号
- SII：Ⅱ道上行出站信号
- S1：1道上行出站信号

### 3. 调车信号机（矮型）
1. D1：下行咽喉调车信号，布置于IIAG区段内
2. D2：上行咽喉调车信号，布置于IIBG区段内

## 五、辅助设备标识
- PZA：安全线旁人工操作按钮，位于1道左侧、5号道岔下方，用于安全线相关人工控制操作
